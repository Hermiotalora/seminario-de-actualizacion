Proyecto - Unidad 1 (JS)
------------------------

Este proyecto contiene la implementacion de los puntos del diagnostico de la Unidad 1 hasta el Punto 7 incluido.

Archivos incluidos:
----------------------
- 01-ref2.html ................ HTML principal para iniciar el sistema
- main.js ..................... Contiene el menu principal y llamadas a funciones
- Application.js .............. CRUD de articulos y funcionalidad de compra (Punto 7)
- APIModelAccess.js ........... Lógica de autenticacion y bloqueo
- LoginApplicationView.js ..... Vista simulada del login

Como probar el sistema:
--------------------------
1. Abri el archivo `01-ref2.html` en cualquier navegador moderno (Chrome, Firefox, Edge)
2. Se pedira iniciar sesion:
   - Usuario: scorpion
   - Contraseña: 987654
3. Una vez autenticado, verás el menu principal.
4. Ingresa a la opción 3 (Gestion de articulos) y luego opcion 5 para comprar articulos.
5. El stock se descuenta tras cada compra exitosa.

Requisitos tecnicos:
-----------------------
- No se necesita servidor ni instalacion. Solo abrir el HTML.
- Todos los scripts estan modularizados con `type="module"` en el HTML.
